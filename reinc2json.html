Reinc2json

Script takes data from 'show reincs' and turns it into json and writes into a file.

Usage:
Type 'rread' to read reincs into array
Type 'rrwrite' to write data into text file, called 'bat_reinc_data_1.json' by default.
Change lines variable to suit your lines setting.
Change FILENAME if needed.

Json data is intended to be used with Reinc analyzer

Get the script from batclient_triggers.
